### Diagnostic hybride de défauts pour un système dynamique

Chaîne de diagnostic combinant modélisation dynamique, observateurs de Luenberger, génération de résidus et machine learning pour la détection, l’isolation et l’estimation de la sévérité des défauts.

Thèmes principaux :

- Systèmes dynamiques
- Diagnostic par observateurs
- Analyse de résidus
- Classification multiclasses
- Correction de défauts capteurs
